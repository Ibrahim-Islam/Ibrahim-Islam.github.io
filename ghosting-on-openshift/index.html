(G)hosting on OpenShift

I have been fascinated with the Ghost Blogging Platform recently and did some probing on it for free on their hosting. I liked it very much and wanted to buy my own hosting but the ones I could afford did not have NodeJS and the ones I could not had. Then I stumbled upon OpenShift by RedHat. I was amazed to see they are providing PaaS with much features for free and started again to probe to host my Ghost blog on OpenShift and I will summarize how I did it below.

  1. After registration, head to Get Started page and select Command Line.

  2. Download and install RubyInstaller and Git For Windows as suggested.

  3. Only after successful completion run gem install rhc followed by gem update rhc and make sure you have to make the path used consistent in the following steps.

  4. Then run rhc setup. Give a unique namespace for all your apps followed by your OpenShift Email and Password to proceed and complete setup.

  5. Now run rhc app create ghost nodejs-0.10 --env NODE_ENV=production --from-code https://github.com/openshift-quickstart/openshift-ghost-quickstart.git which I got from here to create a ghost app on your account. Change ghost name to whatever you like for your blog.

The last step will run script on the given address and setup NodeJS and Ghost along with OpenShift's necessary configuration on your app. Later your app will be cloned on your local machine so that you can make any changes, push it and see the changes immediately.

Give it a shot. It is free! Let me know if you get stuck anywhere.
